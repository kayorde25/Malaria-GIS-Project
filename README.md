# Malaria-GIS-Project
# Spatial Analysis of Malaria Incidence in Nigeria Using QGIS

## Project overview

This project examines the geographic distribution of modelled
Plasmodium falciparum malaria incidence across Nigerian states.

Malaria incidence raster data from the Malaria Atlas Project are
summarised within first-level administrative boundaries using zonal
statistics in QGIS. The resulting state-level estimates are visualised
using a graduated choropleth map.

## Research question

How did estimated Plasmodium falciparum malaria incidence vary across
Nigerian states in 2022?

## Objectives

1. Acquire and document malaria-incidence and administrative-boundary data.
2. Prepare and validate the spatial datasets in QGIS.
3. Calculate state-level malaria-incidence estimates using zonal statistics.
4. Produce a clear and publication-ready choropleth map.
5. Identify important geographic patterns and analytical limitations.
6. Document a reproducible spatial-analysis workflow.

## Geographic scope

Nigeria, first administrative level.

## Indicator

Estimated Plasmodium falciparum malaria incidence per 1,000 population.
